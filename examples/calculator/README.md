# Simple Calculator

This automation performs basic arithmetic operations.

## Input

- `num1`: The first number.
- `num2`: The second number.
- `operation`: The arithmetic operation (add, subtract, multiply, divide).

## Output

- `result`: The result of the operation.

## How to run

```bash
python -m openclaw_automation.cli run 
  --script-dir examples/calculator 
  --input '{"num1": 10, "num2": 5, "operation": "add"}'
```
